# Mission Control: Star Chart

**Mission Control: Star Chart** は、開いているタブ（Stars）をスキャンし、Google NotebookLM の「コックピット」へURLを転送（Beam）するためのChrome拡張機能です。
「宇宙兄弟」のような世界観で、日々の情報収集ミッションをサポートします。

## 機能

- **SCAN**: 現在のウィンドウで開いているタブを自動検出し、リスト（Star Chart）として表示します。
- **SELECT**: NotebookLMに送りたいタブをチェックボックスで選択します。
- **BEAM**: 選択したURLを一括でNotebookLMのチャット入力欄に転送します。
- **ARCHIVE**: 選択したURLリストをローカルストレージに一時保存します（次回以降の機能拡張用）。

## 導入方法

この拡張機能は現在のところ、Chromeウェブストアでは公開されていません。以下の手順で「パッケージ化されていない拡張機能」としてインストールしてください。

1. このリポジトリをZIPでダウンロードして解凍するか、`git clone` してください。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力して移動します。
3. 右上の **「デベロッパーモード」** (Developer mode) をオンにします。
4. 左上の **「パッケージ化されていない拡張機能を読み込む」** (Load unpacked) をクリックします。
5. 解凍（またはクローン）した `MissionControlStarChart` フォルダを選択します。

## 設定方法

特別な設定は不要ですが、頻繁に使用するために以下の設定をおすすめします。

1. Chromeのツールバーにある「パズルピース」アイコン（拡張機能）をクリックします。
2. **"Mission Control: Star Chart"** の横にあるピン留めアイコンをクリックして、ツールバーに常時表示させます。

## 使い方

### 1. 準備
まず、Google Chromeで **[Google NotebookLM](https://notebooklm.google.com/)** のページを開き、ログインしておいてください。
※ NotebookLMのタブが開いていないと、BEAM機能は動作しません。

### 2. スキャンと選択
拡張機能のアイコンをクリックすると、ポップアップウィンドウ（コックピット）が開きます。
自動的に現在のタブがスキャンされ、リスト表示されます。
NotebookLMに送りたいページ（星）にチェックを入れてください。

### 3. BEAM（転送）
**"BEAM"** ボタンをクリックします。
選択されたURLが、開いているNotebookLMのタブに自動的に送信（入力）されます。

> **Note**: 通信エラーが発生した場合は、NotebookLMのページをリロードしてから再度お試しください。
